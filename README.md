
# Clandestino-React

Proyecto de marketplace para gastrobar (Restaurante-Bar)


## Ejecutar localmente

Clonar el proyecto

```bash
  git clone https://github.com/FelixG55/clandestinoReact.git
```
Ir al directorio del proyecto

```bash
  cd clandestinoReact
```
Ir a la carpeta de la API e instalar base de datos

```bash
  cd apiCland/DB
```
Ir al carpeta de la API e instalar dependecia 

```bash
  cd apiCland
```
```bash
  npm install
```
Iniciar API

```bash
  npm run test
```

Ir al directorio del proyecto

```bash
  cd clandestinoReact
```

Instalar dependencias 

```bash
  npm install
```

Iniciar apliacion

```bash
  npm run dev
```


## Tech Stack

**Client:** React, CSS, Bootstrap

**Server:** Node, Express
